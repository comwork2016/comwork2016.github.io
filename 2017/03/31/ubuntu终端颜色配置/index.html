一个单独的颜色设置:\033[代码;前景;背景m，如：\[\033[1;32;40m\]表示高亮显示字体为绿色，背景色为黑色。

注意：颜色的设置，放在相应的要设置的前面，如用户名颜色设置：\[\033[01;35;40m\]\u
